# 🐕 API-Dog

Une application **React** simple et esthétique qui affiche des images aléatoires de 108 races de chiens grâce à l’API publique Dog CEO.  
L’utilisateur peut filtrer les races selon un mot-clé (ex : *husky*, *labrador*, *bulldog*).

---

## 🚀 Fonctionnalités

- 🔍 Récupère la liste complète des races via l’API Dog CEO  
- 🖼️ Affiche une image aléatoire pour chaque race  
- 💬 Filtrage en temps réel grâce à une barre de recherche  
- 💫 Interface moderne, responsive et animée  
- ⚠️ Gestion complète du chargement, des erreurs et de l’affichage  

---

## 🧠 Technologies utilisées

- ⚛️ **React** (Hooks `useEffect`, `useState`)  
- 🌐 **Fetch API** pour la communication HTTP  
- 🎨 **CSS moderne** : grid, flexbox, dégradés, effets d’ombre  

---

## 🌐 API utilisée

**Dog CEO API**
Documentation : [https://dog.ceo/dog-api/](https://dog.ceo/dog-api/)

- Liste des races : `https://dog.ceo/api/breeds/list/all`
- Image aléatoire par race : `https://dog.ceo/api/breed/{breed}/images/random`
